To solve this problem, we need to determine if each word in a given dictionary can be rearranged such that it becomes the lexicographically smallest word compared to all other words in the dictionary. The solution involves checking if each word can be rearranged to be smaller than the maximum possible rearrangement of every other word.

### Approach
1. **Sorting Words**: For each word, compute its minimum and maximum possible lexicographical order by sorting the characters in ascending and descending order, respectively.
2. **Max List**: Create a list of maximum possible rearrangements (sorted in descending order) for all words.
3. **Determine Minimum Values**: Compute the smallest and second smallest values from the list of maximum rearrangements. This helps in efficiently determining if a word can be rearranged to be the smallest.
4. **Check Feasibility**: For each word, check if its minimum possible rearrangement is smaller than the smallest maximum rearrangement of all other words. This involves comparing the minimum rearrangement of the current word against the smallest possible maximum rearrangement of the remaining words.

### Solution Code
```python
def main():
    import sys
    input = sys.stdin.read().split()
    n = int(input[0])
    m = int(input[1])
    words = input[2:2+n]
    
    if n == 1:
        print('1')
        return
    
    sorted_mins = []
    sorted_maxs = []
    for word in words:
        sorted_word = ''.join(sorted(word))
        sorted_mins.append(sorted_word)
        sorted_maxs.append(''.join(sorted(word, reverse=True)))
    
    max_list = sorted_maxs
    sorted_max_list = sorted(max_list)
    
    first_min = sorted_max_list[0]
    count_first_min = 0
    for s in sorted_max_list:
        if s == first_min:
            count_first_min += 1
        else:
            break
    
    if count_first_min == len(sorted_max_list):
        second_min = first_min
    else:
        second_min = sorted_max_list[count_first_min]
    
    res = []
    for i in range(n):
        original_max_i = max_list[i]
        if original_max_i != first_min:
            current_min = first_min
        else:
            if count_first_min > 1:
                current_min = first_min
            else:
                current_min = second_min
        if sorted_mins[i] < current_min:
            res.append('1')
        else:
            res.append('0')
    
    print(''.join(res))

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read and split into individual components for processing.
2. **Special Case Handling**: If there's only one word, it trivially satisfies the condition and returns '1'.
3. **Sorting Words**: Each word is sorted to determine its minimum and maximum possible lexicographical order.
4. **Max List and Sorting**: The maximum possible rearrangements of all words are stored in a list, which is then sorted to determine the smallest and second smallest values.
5. **Feasibility Check**: For each word, check if its minimum possible rearrangement is smaller than the smallest possible maximum rearrangement of the other words. This is done by comparing against the smallest and second smallest values from the sorted list of maximum rearrangements.

This approach ensures that we efficiently determine if each word can be rearranged to be the smallest, leveraging sorting and comparison techniques to handle large input sizes effectively.
