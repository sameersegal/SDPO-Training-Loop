To solve this problem, we need to determine if each word in a dictionary can be rearranged such that it is the lexicographically smallest among all other words after any number of character swaps. The key insight is to compare the sorted version of each word with the minimum possible reverse-sorted version of other words.

### Approach
1. **Sort Each Word**: For each word, compute its lexicographically smallest form (sorted in ascending order) and its lexicographically largest form (sorted in descending order).
2. **Find Minimum Reverse-Sorted Values**: Identify the minimum reverse-sorted value across all words. This helps in determining the smallest possible largest value of any other word.
3. **Determine Second Minimum**: If the minimum reverse-sorted value appears multiple times, the second smallest value is used to handle cases where the minimum value is the same for multiple words.
4. **Check Feasibility**: For each word, check if its sorted form is smaller than the minimum reverse-sorted value of all other words. This ensures that the word can be rearranged to be the smallest.

### Solution Code
```python
import sys

def main():
    input = sys.stdin.read().split()
    n = int(input[0])
    m = int(input[1])
    words = input[2:2+n]
    
    # Compute sorted and reverse_sorted for each word
    sorted_list = []
    reverse_sorted_list = []
    for word in words:
        sorted_word = ''.join(sorted(word))
        reverse_sorted_word = ''.join(sorted(word, reverse=True))
        sorted_list.append(sorted_word)
        reverse_sorted_list.append(reverse_sorted_word)
    
    # Find overall_min and count
    overall_min = min(reverse_sorted_list)
    count = reverse_sorted_list.count(overall_min)
    
    # Find second_min
    second_min = None
    for s in reverse_sorted_list:
        if s != overall_min:
            if second_min is None or s < second_min:
                second_min = s
    
    # For each word, check
    result = []
    for i in range(n):
        current_reverse_sorted = reverse_sorted_list[i]
        if current_reverse_sorted != overall_min:
            min_for_j = overall_min
        else:
            if count > 1:
                min_for_j = overall_min
            else:
                min_for_j = second_min
        if sorted_list[i] < min_for_j:
            result.append('1')
        else:
            result.append('0')
    
    print(''.join(result))

if __name__ == "__main__":
    main()
```

### Explanation
1. **Sorting**: Each word is sorted to find its smallest possible form and reverse-sorted to find its largest possible form.
2. **Finding Minimum Reverse-Sorted Value**: The minimum value among all reverse-sorted words is determined to check if a word can be the smallest.
3. **Handling Multiple Minimums**: If the minimum reverse-sorted value occurs multiple times, we ensure the check remains valid by using the second smallest value when necessary.
4. **Comparison**: For each word, we check if its sorted form is smaller than the minimum reverse-sorted value of other words, ensuring it can be rearranged to be the smallest.

This approach efficiently checks each word against the necessary conditions using sorting and comparison, ensuring optimal performance even for large input sizes.
